CSS 규칙
작은 따옴표 사용 범위
  • 폰트의 선언, filter, content에 작은 따옴표('')를 사용한다. 그 외의 경우는 사용하지 않는다.
.class_name {font-family:'dotum'} .class_name:after{content:'.'} .class_name {background:url(img.gif) no-repeat}
세미콜론
  • 마지막 선언된 속성에도 세미콜론(;)을 사용한다.
공백의 사용
  • 선택자 간, 중괄호간 한칸의 공백을 사용한다, 중괄호 안쪽 좌우에는 공백은 사용하지 않는다.
.class_name, .class_name .child_class {background:url(../img/img.gif) no-repeat;} .class_name {padding:0; margin:0;} .class_name {padding:0; margin:0;}
들여쓰기 줄바꿈
  • CSS 코드 작성시 들여쓰기는 허용하지 않는다.
  • 공통으로 사용되는 속성의 선택자간의 줄 바꿈은 허용하며 이 외의 줄바꿈은 허용하지 않는다.
h4.class_name {font-size:1.25em; letter-spacing:-1px; margin:0.5em 0 0 0;} .class_name, .class_name .child_class {background:url(../img/img.gif) no-repeat;}
단위
  • 모바일과 같이 유동적인 레이아웃 구현 시 상대단위(em, rem, %)를 사용한다.
  • 1px과 text-indent:-9999em, border 단위는 예외로 한다.
z-index
  • z-index 값은 최소 10부터 시작하며 최고 9999이며 10 단위로 증감한다. 단, 중간에 변수가 생기면 예외로 1단위 값을 지정할 수 있다.
구분 단위 설명
인라인 1 우선순위 1씩 증가, 기본요소 0
그룹 10 우선순위 1씩 증가, 기본요소 10
구조 100 우선순위 1씩 증가, 기본요소 100
팝업 1000 우선순위 1씩 증가, 기본요소 1000
주석 / 빈줄
  • 종료 주석은 작성하지 않는다.
  • 코드 그룹 간 1줄의 빈 줄을 허용한다. 단, 1줄을 초과하지 않는다.
/* title */ .title_type1 {margin:30px 0 10px 0;} /* description */ .desc1 {padding:2px 0 0 15px; font-size:0.917em;}
속성선언순서
  • 밴더속성과 핵속성은 해당 속성 뒤에 선언한다.
    예) .btn {border:1px solid #f60; *border:2px solid #f60; border-radius:2px; -webkit-border-radius:2px}
  • !important 사용은 지양한다.
순서 구분 속성
1 Layout display, visibility, overflow
2 위치 position, float, clear, z-index
3 크기/간격 width, height
margin, padding
4 모양/배경 border
background
5 폰트/정렬 font, color, text-align, vertical-align
6 기타 언급되지 않은 속성들은 후순위로 적용한다.
큰글씨 설정
  • 조회(일부), 이체 카테고리에서만 적용한다.
  • 큰글씨모드는 최상위 선택자에 .big_mode 클래스를 선언한다.
html.big_mode {font-size:11px !important;}
다크테마 설정
  • 기본 color값은 :root 에 변수처리 한다.
    예) background:var(--fc_11, #fff);
  • 변수명은 일반테마/ 다크테마 동일하며, color값만 변경한다.
  • 다크테마는 최상위 선택자에 .dark_mode 클래스를 선언한다.
  • 다크테마 관련 예외케이스는 각 CSS 제일 하단에 추가한다.
/* common.css */ :root{ --fc_01: #000; --fc_02: #26282c; --fc_03: #484b51; --fc_04: #696e76; --fc_05: #8c949e; --fc_06: #aab0b8; --fc_07: #c6cbd0; --fc_08: #dde1e4; --fc_09: #ebeef0; --fc_10: #f7f7f8; --fc_11: #fff; --bg_01: #fff; /* Contents BG */ --bg_02: #f7f7f8; /* Contents Notice */ --bdc_01: #ebeef0; /* Line Default */ --bdc_02: #696e76; /* Line Design Type */ --bdc_03: #c6cbd0; /* Data List Header */ --bdc_04: #484b51; /* Accordion Open Header */ } :root.dark_mode{ --fc_01: #000; --fc_02: #fff; --fc_03: #f7f7f8; --fc_04: #ebeef0; --fc_05: #dde1e4; --fc_06: #c6cbd0; --fc_07: #aab0b8; --fc_08: #8c949e; --fc_09: #696e76; --fc_10: #484b51; --fc_11: #26282c; --bg_01: #212224; --bg_02: #34363a; --bdc_01: #45484d; --bdc_02: #86888a; --bdc_03: #65696e; --bdc_04: #8c8c8e; }
