$('#alert_btn').on('click', function(){

        titleUse:false,
        content:'#alert_style_html',

    });

- Description
- alert형 팝업 : 별도의 html없이 Javascript 옵션에서 지정만으로 구현
- alert 대체 팝업 : 팝업 내용 마크업 필요
1. 버튼 ID로 click 이벤트 생성
2. $.ohyLayer() 함수 호출
3. $.ohyLayer() 함수 내 옵션 작성

- Option


    
        

            titleUse
            true / false
            타이틀 사용 여부 (default: true)
        

        

            title
            문자열
            컨펌창 타이틀
        

        

            content
            문자열
            알림창일 경우 텍스트 입력 ('알림창입니다.')
        

        

            type
            alert / confirm alert
            레이어 타입 지정
        

        

            closeUse
            true / false
            오른쪽 상단 닫기버튼 사용여부 (default: true) (X버튼에는 {content}_dev id가 자동 생성됨)
        

        

            closeAct
            true / false
            닫기 버튼의 동작 사용 유무 (default: true)
        

        

            dimAct
            true / false
            dim 클릭 시 팝업 닫기 사용 유무 (default: false)

- JavaScript
//변수 레이어 시작
var _html = $.ohyLayer({
            openAuto : true,
            title:'컨펌 레이어',          
            content:'#htmlTest_var',
            type:'bottom',
            closeUse:true,
            closeAct:false,
        });
//변수 레이어 열기
$("#layer_var").on("click", function(){
    _html.open();
});

//변수 레이어 닫기
$(document).on("click", "#htmlTest_var_confirm_close", function(){
    _html.close();
});

$(document).on('click', "#htmlTest_var_dev", function(){
    $.ohyLayer({
        title:'컨펌 레이어',          
        content:'#htmlTest_var_confirm',
        type:'confirm',
        closeUse:false,
    });
});
//변수 레이어 끝

- Description
- 팝업 내용 마크업 필요
1. $.ohyLayer()함수 작성 후 {변수}에 할당, 옵션작성
2. 버튼 ID로 click 이벤트 생성, {변수}.open(); 으로 $.ohyLayer()함수 실행
3. 팝업 닫기 : {변수}.close()
