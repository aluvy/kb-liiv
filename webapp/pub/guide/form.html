필수입력항목
- .form_item.req 클래스를 함께 준다.
- 필수입력항목 내 input에 blur event가 일어날 경우 밸류값을 체크하여 _input.req()함수 실행
   => error 클래스가 추가됨 .form_item.req.error(안내문구가 보여짐) / .input.error(빨간색 밑줄 표시)

error
- input에 blur event가 일어날 경우 밸류값이 "error"(임시값)일 경우 _input.error()함수 실행
   => error 클래스가 추가됨 .form_item.req.error(안내문구가 보여짐) / .input.error(빨간색 밑줄 표시)

버튼추가
- .item_content에 .btn_check or .btn_address or .btn_search 등등(추가예정) 클래스를 추가하면 자동으로 버튼이 생성됨
