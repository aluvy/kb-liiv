    상세내용

    • 데이터 같이쓰기 결합 시 매월 1일에 결합 상대 회선(데이터 받기 요금제)으로부터 데이터 2GB 제공됩니다. (데이터 주기 요금제의 데이터 기본 제공량에서 2GB 차감
    • 데이터 주기요금제와 받기요금제 간 결합시, 결합 시점에 한정하여 데이터 주기요금제의 데이터 차감없이 받기요금제에 프로모션으로 데이터 2GB 제공하고, 익월(1일)부터 주기요금제의 데이터 기본제공량 2GB 차감, 받기요금제에 2GB 충전

    서비스 이용방법

    • 데이터 같이쓰기 결합이 가능한 요금제인지 확인 (매월 2GB를 받을 회선에서도 결합 신청 가능)
    • 데이터 같이쓰기를 신청할 회선 정보(이름, 휴대폰번호)를 넣고 신청하면 결합할 상대방 회선으로 인증 문자 발송
    • 상대방에게 인증번호를 받아 인증 완료 시 결합 완료

    유의사항

    • 데이터 같이쓰기 결합 분리 후 3개월 이내 재결합 불가 (3개월 이내 결합 서비스 가입 이력이 있는 경우 재가입 불가)
    • 결합은 KB Liiv M KT망 내 데이터 같이쓰기 결합 가능한 요금제 내 동일명의 결합 및 타인 결합 가능
    • 결합해지는 주기요금제와 받기요금제 중 한 쪽에서 해지하면 분리되며, 해지 익월부터 주기요금제 데이터 차감 및 받기요금제 데이터 충전이 중단됨
    • 데이터 같이쓰기 결합은 전용 부가서비스 가입을 통해 이루어 집니다. (LTE결합서비스_데이터받기2GB/데이터주기2GB)
    • 데이터 같이쓰기 결합은 특정 요금제에서만 가입 가능합니다. (결합 가능 요금제 리스트는 아래 참고)
