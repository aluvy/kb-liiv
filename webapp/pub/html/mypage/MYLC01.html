요금 납부 방법

결제계좌
계좌번호

본인 명의의 계좌정보를 입력해주세요.

  • 계좌 납부 시, 자동 납부 진행됩니다.
  • 당행계좌(KB국민은행)로 자동이체 신청 시 자동납부 신청결과가 익일(은행 영업일 기준) 이내 반영되고, 타행계좌(KB국민은행 외 타은행 또는 증권계좌)로 신청 시 2영업일 (은행영업일 기준) 이내 반영되므로 정상적으로 자동납부 등록되었는지 추후에 반드시 확인하시기 바랍니다.
  • SKT망의 경우 5일 이후 자동납부 변경시 익월 정기분부터 반영됩니다.
  • 타인명의의 계좌나 카드로 납부방법의 변경을 원하시는 경우 KB Liiv M 고객센터에서 명의자 확인 후 변경 가능합니다.

유효성 check
  • 인증서가 없는 경우, 인증서 발급 후 인증을 진행해주세요.
  • 인증서 발급이 불가하신 경우 신용카드로 자동납부 신청 후개통 익일 KB Liiv M 고객센터 : 114(무료), 1522-9999(유료)를 통해 계좌이체로 변경이 가능합니다.

카드번호
유효기간

본인명의의 신용/체크카드 정보를 입력해주세요

  • 카드 납부 시, 자동 납부 진행됩니다.
  • 카드로 자동이체 신청시 자동납부 결과가 익일(은행 영업일 기준)에 반영되므로 정상적으로 자동납부 되었는지 추후에 반드시 확인하시길 바랍니다.
  • SKT망의 경우 5일 이후 자동납부 변경시 익월 정기분부터 반영됩니다.
  • 자동납부 카드 등록은 카드 실물 수령 후 정상 등록이 가능합니다.
  • 자동 납부 카드 변경은 개통완료 후 「마이페이지 > 고객정보 조회 > 납부방법 변경」에서 처리 가능합니다.
  • 타인명의의 계좌나 카드로 납부방법의 변경을 원하시는 경우 KB Liiv M 고객센터에서 명의자 확인 후 변경 가능합니다.
