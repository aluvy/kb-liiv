    기타

    • U+ ZONE 무료제공 : 전국에 설치된 U+ 무선랜 공유기를 하나의 네트워크로 연결하여, 무선인터넷 이용가능합니다.
    • VoLTE 통화가능 단말의 경우 VoLTE 통화가 자동 적용됩니다.
    • 테더링 이용(테더링 : 내 휴대폰과 연결하여 다른 기기에서 와이파이 사용)
      • 테더링은 기본 제공데이터 한도내에서 이용가능하며 기본 제공데이터를 모두 사용하면 테더링 이용이 차단됩니다.
      • 든든 무제한 요금제는 기본 제공데이터 소진 후 속도제어(1Mbps 또는 3Mbps)시 테더링 이용이 불가합니다.
    • 데이터쉐어링에 사용되는 데이터는 요금제별 데이터 월 제공량 내에서 합산하여 차감되며, 월 제공량 소진 시 자동차단됩니다.
      • 단, 든든 1GB, 든든 무제한1GB+ 요금제는 데이터쉐어링은 사용이 불가능 합니다.
      • 든든 무제한 5GB+/7GB+ 요금제는 월제공량 (5GB/7GB)내에서 합산하여 차감되며, 월 제공량 소진 후 데이터쉐어링 사용시 별도 과금됩니다.
      • 든든 무제한 11GB+(신) 요금제는 월 제공량(11GB) 및 일 제공량(2GB) 내에서 합산하여 차감되며, 월 제공량 및 일 제공량 소진 후 데이터쉐어링 사용 시 별도 과금됩니다.
    • mVoIP(데이터 음성통화 : 보이스톡)
      • 요금제별 기본 제공 데이터 한도 내에서 해당 요금제의 mVoIP 허용량 만큼 mVoIP서비스 이용이 가능합니다.
      • 든든 무제한 요금제는 데이터 허용량 제한없이 mVoIP서비스 이용이 가능합니다. (단, 데이터 속도가 1Mbps 또는 3Mbps로 제어되는 경우 서비스 연결이 원활하지 않을 수 있습니다.)
      • 통화연결로 발생하는 데이터는 소모될 수 있으며 Wi-Fi망에서는 요금제와 상관없이 사용 가능합니다.
