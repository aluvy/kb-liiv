  • div.form_item.fix
    .label 영역이 fix 된다
  • div.form_item.req
    필수입력 사항으로 input에 value 값이 없는 상태에서 blur 이벤트가 발생할 경우 에러메세지가 노출된다.
    label우측에 * 표시
  • div.item_content.focus
    input에 focus 이벤트가 일어난 상태
  • div.item_content.done
    input에 value값이 입력되고 blur 이벤트가 일어난 상태
  • div.item_content.error
    input에 입력한 value값이 잘못되고, blur 이벤트가 일어난 상태
  • div.item_content.btn_xxx
    btn_xxx 클래스명에 따라 자동으로 버튼이 삽입된다.
  • div.item_content.js_timer
    인증번호 입력 시 제한시간 timer 영역이 자동으로 삽입된다. (MALG0108 참고)
