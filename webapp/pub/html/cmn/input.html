  • div.form_group
    • 폼 영역을 싸고있는 div
  • div.form_item (.fix .req .error)
    • .fix
      • .label 영역이 fix 된다
    • .req
      • 필수 입력사항으로 input에 value값이 없는 상태에서 blur 이벤트가 발생할 경우 에러메세지가 노출된다.
      • label 우측에 * 표시
    • .error
      • 에러메세지 영역 노출됨 (.form_group .form_item.error ~ .form_error)
  • div.item_content (.focus .done .error) (.btn_xxx .js_timer)
    • .focus
      • input에 focus 이벤트가 일어난 상태
      • 검은색 2px 라인 표시됨 (.form_group .item_content::after)
    • .done
      • input에 value값이 입력되고 blur 이벤트가 일어난 상태
    • .error
      • 빨간색 2px 라인 표시 됨 (.form_group .item_content.error::after)
    • .btn_xxx
      • btn_xxx 클래스명에 따라 자동으로 버튼이 삽입된다.
    • .js_timer
      • 인증번호 입력 시 제한시간 timer 영역이 자동으로 삽입된다. (MALG0108 참고)
  • div.label (.blind)
    • .blind 접근성을 위해 대체텍스트 처리 필요 시 사용
