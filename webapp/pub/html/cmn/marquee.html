Marquee 마크업 구조

#marquee_area (호출문 옵션 content 에 넣을 ID)

.marquee-wrap (left 값을 받을 엘리먼트)

.marquee-item (clone 될 엘리먼트)

  • CSS스타일이 맞지 않으면 제대로 동작하지 않음
  • #marquee_area
    호출문 옵션 중 content에 넣을 ID
  • .marquee-wrap
    • 고정 class
    • position:absolute 를 가짐
    • left 값을 받아 이동할 엘리먼트
  • .marquee-item
    • 고정 class
    • clone이 될 엘리먼트

Marquee 호출문

$.marquee({
    content:'#marquee_notice'
});
  • content
    marquee 애니메이션 적용할 엘리먼트 ID
  • speed
    숫자 (default: 1)
    애니메이션 속도
  • playUse
    true / false (default:true)
    로딩 시 애니메이션 시작 여부
  • clone
    숫자 (default: 2)
    클론 갯수

STOP / PLAY / CLOSE 버튼

  • #marquee_area_stop
    • marquee 애니메이션 멈춤
  • #marquee_area_play
    • marquee 애니메이션 멈춘 후 play
  • #marquee_area_close
    • instance가 삭제되며 다시 play할 수 없음
