달력 팝업 호출하기

_calendar.open( 'idCal', 'idInput', '20230223', '-', { 'minDate':20230302, 'maxDate':20230317 } );

@param

  • idCal : 생성할 달력 팝업의 ID
  • idInput : 선택한 날짜를 입력받을 input ID
  • 20230223 : 기본 날짜설정
    • ex) '' : 공백으로 처리하면 오늘날짜가 기본으로 선택됨
    • ex) $(document).find("#idInput").val() : 과 같은 방법으로 사용가능
  • - : 입력한 문자열로 년/월/일이 구분됨 (보내지 않으면 기본 '-' )
    • ex) '' : 00001122 (구문문자 없음)
    • ex) '.' : 0000.11.22
    • ex) '/' : 0000/11/22
  • { 'minDate':20230302, 'maxDate':20230317 } : minDate, maxDate 설정
    • 형식에 맞지 않는 데이터를 보낼 경우 작동 안함
    • ex) { 'minDate':20230302 } : minDate만 설정 가능
    • ex) { 'maxDate':20230317 } : maxDate 설정 가능
