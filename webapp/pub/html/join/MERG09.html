  • 유의사항
    • LTE 데이터 전용 태블릿 등으로 데이터를 나눠쓸 수 있습니다.
    • 데이터쉐어링은 유심이 장착 가능한 태블릿에서만 이용할 수 있습니다.
      • 휴대폰끼리는 데이터쉐어링을 할 수 없습니다. (KT망은 스마트폰 가능)
    • 데이터쉐어링 요금제는 휴대폰 회선이 LTE 휴대폰과 LTE 요금제만 서비스 가능합니다.
    • LTE 요금제 가입 명의자와 같은 명의로 가입한 기기로만 데이터를 나눠쓸 수 있습니다.
    • 모바일 1개 요금제 당 1개 데이터쉐어링 요금제 결합 가능합니다.
    • 데이터쉐어링 요금제는 단독 사용은 불가하며, 휴대폰 회선과 데이터쉐어링 결합(연결) 시에만 사용이 가능합니다.
    • 신청 및 이용방법 : 데이터쉐어링 요금제(자회선) 신규 개통 > 휴대폰 및 태블릿 확정기변 > 데이터쉐어링 연결
      • LG U+망 : 확정기변을 하지 않고 비정상적으로 데이터쉐어링 사용시 쉐어링 서비스 이용을 할 수 없습니다.
      • KT망 : 유심을 데이터쉐어링 이용단말에 인식후 사용
    • 데이터쉐어링 요금제(자회선)는 번호이동이 불가합니다. 신규로 신청 바랍니다.
    • 쉐어링으로 연결한 요금제(자회선)는 음성/영상/문자 착발신 불가, 문자는 가능 단말에 한하여 수신만 가능합니다.
      • 잔여량/초과 사용량 SMS 알림번호 등록/변경을 하면, 문자를 수신할 수 없는 단말의 사용량 정보를 등록한 휴대폰에서 대신 받아 볼 수 있습니다.
    • 데이터쉐어링이 가능한 요금제
      • LG U+망 : LTE 든든 3GB, LTE 든든 5GB, LTE 든든 7GB, LTE 든든무제한 5GB+, LTE 든든무제한 11GB+, The주는 LTE 15GB+, Flex LTE, 나라사랑 LTE, 국민든든 LTE, 무궁화/공무원든든/선생님든든 LTE, 반려행복 LTE, LTE 든든실속 5GB, LTE 든든실속 10GB, LTE 든든실속 7GB+, 청년희망 LTE, 청년희망 LTE(Ⅱ)
      • KT망 : LTE실속 15GB+(100분)(KT망), LTE실속 15GB+(300분)(KT망), LTE 무제한 7GB+(KT망), LTE무제한 11GB+(KT망), LTE무제한 11GB+(왓챠)(KT망), LTE무제한 100GB+(시즌)(KT망), LTE무제한 100GB+(게임박스)(KT망), LTE 무제한 일5GB+(시즌)(KT망)
    • 쉐어링 요금제(모회선)의 쉐어링 가능 용량 소진 시 데이터는 차단되며, 쉐어링과 합산하여 데이터가 차감됩니다.
      • LG U+망
        : LTE 든든 3GB~7GB 요금제의 경우 월기본제공량 내에서 데이터쉐어링이 가능합니다.
        : Flex LTE 요금제는 월 11GB 내에서 데이터쉐어링이 가능합니다.
      • KT망
        : 데이터쉐어링에 사용되는 데이터는 기본제공 데이터(월 제공량 + 일 제공량) 내에서 합산하여 차감됩니다. 기본제공 데이터 소진 시 데이터쉐어링은 200Kbps 속도로 지속 이용 가능합니다.
    • 쉐어링 요금제(모회선)의 최대 속도제어(QoS)로 데이터 사용시 무과금 처리되나, 연결한 요금제(자회선)에서는 데이터 사용시 별도 과금 처리됩니다. (QoS는 모회선만 적용됨)
      • LG U+망 : LTE 든든무제한 5GB+, LTE 든든무제한 11GB+, The주는 LTE 15GB+, Flex LTE, 나라사랑 LTE, 국민든든 LTE, 무궁화/공무원든든/선생님든든 LTE, 반려행복 LTE, LTE 든든실속 7GB+, 청년희망 LTE 요금제와 같이 속도제한으로 데이터 무제한 사용이 가능한 요금제는 기본 제공 데이터 소진 후 데이터 사용 시 모회선에서는 무과금 처리되나, 자회선에서는 모회선의 기본 제공 데이터 소진 후 초과 사용 시 별도 과금 처리됩니다.
    • 서비스 해지 후 1개월 이내 재가입이 불가합니다.
    • 법인 고객은 가입이 불가합니다.
    • 쉐어링으로 연결한 요금제(자회선)는 테더링 및 해외로밍 이용이 불가합니다.
    • 신청 및 이용방법은 모바일웹 및 고객센터에서 확인 가능합니다.
    • USIM 발급비용은 1회선 당 1회, 고객별 최대 3회까지 면제됩니다. 면제횟수 초과시 유심발급비(배송비 포함) 7,700원이 다음달 청구됩니다.
