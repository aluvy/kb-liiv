셀프개통 가능 시간

신규개통
08:00 ~ 20:00 (365일)
번호이동
10:00 ~ 20:00
(일요일, 공휴일 제외)
  • 신규개통 동영상 가이드
  • 번호이동 동영상 가이드

신규개통

  1. KB Liiv M 모바일웹페이지에 접속하세요.

    아이디/비밀번호 로그인
    (영업점 가입 고객은 최초 로그인 시 비밀번호 설정 필요)

  2. 신청된 내용을 확인하고 개통을 진행하세요.

    • 가입신청 정보 확인은 2-1,
      또는 2-2 화면에서
      '신청 현황 확인' 클릭

    • 바로개통 희망 시, 2-2
      화면에서 '개통하기' 클릭
      (2-3 화면으로 바로 이동)

    • 신청정보 확인 후
      '개통신청' 클릭

    • 개통 희망 시,
      '개통하기' 클릭

  3. 사용 희망번호를 검색하고
    선택해주세요

  4. 유심 일련번호를 입력해주세요

    통신망에 맞게 유심 일련번호 자리수를
    확인 후 입력해주세요.

  5. 개통신청 확인까지
    누르면 완료!

  6. 개통이 완료된 유심을
    휴대폰에 삽입 후 이용하세요.

    • 휴대폰 유심 사이즈에
      맞게 유심칩을 떼어 장착

    • 유심 장착 시 안내문구가
      위를 향하도록 장착

번호이동

  1. KB Liiv M 모바일웹페이지에 접속하세요.

    아이디/비밀번호 로그인
    (영업점 가입 고객은 최초 로그인 시 비밀번호 설정 필요)

  2. 신청된 내용을 확인하고 개통을 진행하세요.

    • 가입신청 정보 확인은 2-1,
      또는 2-2 화면에서
      '신청 현황 확인' 클릭

    • 바로개통 희망 시, 2-2
      화면에서 '개통하기' 클릭
      (2-3 화면으로 바로 이동)

    • 신청정보 확인 후
      '개통신청' 클릭

    • 개통 희망 시,
      '개통하기' 클릭

  3. '번호인증'을 진행해주세요

    은행 계좌번호, 신용카드 번호, 휴대폰 일련번호 중
    인증방법 한가지를 선택

    필수

    • '인증번호' 확인 버튼
      클릭하면 인증완료

    • 번호이동 사전동의 정보
      확인후 '동의 클릭'
      할인반환금 확인(요금 및 휴대폰)

    '번호이동' 사전동의 진행방법
    수신된 문자 URL을 통해 인증 (3-1 ~ 3-3 화면참고)
    ARS 인증을 통해 인증 후 4-1 화면으로 이동
    (통신사 별 ARS번호로 전화하여 진행)
    • 휴대전화번호 뒤 4자리가 아님에 유의하시기 바랍니다.

  4. 유심 일련번호를 입력해주세요

    통신망에 맞게 유심 일련번호 자리수를
    확인 후 입력해주세요.

  5. 개통신청 확인까지
    누르면 완료!

  6. 개통이 완료된 유심을
    휴대폰에 삽입 후 이용하세요.

    • 휴대폰 유심 사이즈에
      맞게 유심칩을 떼어 장착

    • 유심 장착 시 안내문구가
      위를 향하도록 장착

  • 유의사항
    • 유심은 반드시 개통완료 후에 장착해주세요.
    • 서비스 이용이 원활하지 않은 경우, 휴대폰을 2~3회 재부팅 해주세요.
    • 유심은 꼭 개통 완료 후 휴대폰에 장착해주세요.
    • 셀프개통 작업이 완료될 때 까지 화면을 닫지 마시고 개통 결과를 확인해주세요.
    • 개통이 완료되었으나, 실제로 휴대폰 사용이 불가한 경우에는 KB Liiv M 고객센터(1522-9999)에 문의해주세요.
